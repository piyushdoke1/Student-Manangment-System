# Student Management System

A web-based Student Management System developed using Python and Flask.  
The application allows users to manage student records efficiently with CRUD operations.

## Features

- Add student records
- Update student details
- Delete student data
- View all students
- SQLite database integration
- Simple and responsive UI

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
